# CyberHand-AR ✋⚡

CyberHand-AR is a real-time hand tracking augmented reality (AR) web application built using MediaPipe and JavaScript. It transforms hand gestures into an interactive visual and audio experience with dynamic neon effects.

## 🚀 Features

- ✋ Real-time hand tracking using MediaPipe
- 🎯 Gesture recognition (pinch, open hand, fist)
- 🌈 Multiple visual themes (Cyberpunk, Rainbow, Lava, Ocean, Galaxy)
- ⚡ Interactive effects (shockwaves, particles, lightning connections)
- 🎵 Audio-reactive system based on hand movement
- 🌀 Dynamic background animations (matrix-style visuals)
- 📊 Live HUD (FPS, gesture detection, hand count)

## 🧠 Tech Stack

- JavaScript (Vanilla)
- MediaPipe Hands
- HTML5 Canvas
- Web Audio API

## 🎮 How It Works

- Detects hand landmarks using MediaPipe
- Tracks finger movements in real time
- Converts gestures into visual and audio effects
- Creates immersive AR interaction directly in the browser

## 🛠️ Setup Instructions

1. Clone the repository
2. Open with Live Server (VS Code recommended)
3. Allow camera permissions
4. Click "Enter Experience"
